Detailed Description

A Feature that represents a grouping of related features or a complex polygon.

Each Relation has one or more member features, which can be retrieved via members(). Each Feature object returned by members() has its role set to a non-empty string, if it has an explicit role in this Relation (e.g. "stop" for a Node in a route relation that represents a bus stop).

Note: Functions that require the complete geometry of a Relation (such as area() or centroid()) may fail with a QueryException if one or more tiles that contain members of the Relation are missing.
